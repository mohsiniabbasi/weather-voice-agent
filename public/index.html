Weather-Aware Voice Agent

Call in, ask for the weather in any town. If it's below 10 °C, you get a "bring a coat" reminder. This is the serverless API behind the voice agent — a functions-only deployment, so there's no app to click through here.

POST/api/weather

Mid-call path. Geocodes the town, fetches the current temperature, applies the <10 °C rule, sends a reminder if cold, and returns a spoken sentence for the agent.

POST/api/call-webhook

Post-call path. Idempotent call logging (dedupes on call_id) with a dead-letter queue for failed writes.

GET/api/reconcile

Missing-webhook detection. Flags calls that started but never received a call_ended event.

Retell (voice) → Vercel serverless functions → Open-Meteo (weather) → Twilio SMS, email fallback → Upstash Redis (state).  ·  Source on GitHub
